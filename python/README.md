# REST API in Python

Create , List , Update APIs for User in Python(Django)

## Installation

Use the package manager [PIP3](https://pip.pypa.io/en/stable/).

```bash
pip3 install -r requirements.txt
```

## Execution 

Use Python3 and pip3

```bash
python3 manage.py runserver
```
